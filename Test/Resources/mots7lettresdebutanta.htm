Vous aimez jouer au scrabble, nous vous proposons trois sites web : www.BestWordClub.com, www.Ortograf.com et www.Jette7.com.

  • Avec Jette7, vous pouvez jouer au scrabble avec les mêmes règles qu'en club de scrabble. Le site vous offre quelques parties gratuites, puis le jeu devient payant, soit quelques sous par partie.
  • Le jeu Ortograf vous permet de jouer au scrabble contre votre ordinateur Macintosh (4 niveaux de difficulté). Le jeu est facile à utiliser et vous procurera des heures de plaisir. Vous pouvez jouer gratuitement avec la version démo (sans les E).
  • Le jeu BestWordClub vous permet de jouer au scrabble en mode duplicate. C'est-à-dire avec plusieurs personnes à la fois sur la même grille, avec les mêmes lettres. Le jeu est facile à utiliser, vous pouvez y jouer gratuitement tant que vous voulez.
