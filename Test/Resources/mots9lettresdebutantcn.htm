Vous aimez jouer au scrabble, nous vous recommandons 3 sites web : www.Ortograf.com, www.Jette7.com et www.BestWordClub.com.

  • Le jeu Jette7 vous permet de jouer au scrabble exactement comme si vous étiez dans un club de scrabble. Vous pouvez jouer quelques parties gratuites, après il vous en coûtera quelques centimes par partie.
  • Avec BestWordClub, vous pouvez jouer au scrabble en mode duplicate. C'est-à-dire avec plusieurs personnes à la fois sur la même grille, avec les mêmes lettres. Le jeu est facile à utiliser, vous pouvez y jouer gratuitement tant que vous voulez.
  • Le jeu Ortograf vous permet de jouer au scrabble contre votre ordinateur Macintosh (4 niveaux de difficulté). L'interface est simple à utiliser et vous procurera des heures de plaisir. Vous pouvez jouer gratuitement avec la version démo (sans les E).

Vous aimez les mots et les listes de mots, nous vous proposons 2 sites web : 1mot.fr et www.ListesDeMots.com.

  • Entrez n'importe quel mot sur le site web 1mot.fr pour avoir ses infos, ses anagrammes, ses suffixes, ses préfixes, etc.
  • Sur le site web www.ListeDeMots.com, vous trouverez des centaines de listes de toutes les sortes, les mots avec J, les mots avec K, les mots avec Q sans U, les suffixes, les mots se terminant par, etc.
